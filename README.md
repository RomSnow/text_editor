# Текстовый редактор
Автор: Ивашкин Роман (ivashkin.roma2001@yandex.ru)

Описание
---------------------
Редактор для просмотра и редактирования простых текстовых файлов

Требования
---------------------
* Python 3
* PyQt 5

Состав
---------------------
* Запускаемый файл `/text_editor.py`
* Рабочие файлы `graph_editor`

Дополнение
---------------------
Редактор может работать только с простыми текстовыми форматами

Например: .txt, .py, .js, .html, .css и тд
